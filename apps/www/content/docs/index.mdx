## What is mono-ui?

mono-ui is a collection of reusable components built on top of [Radix UI](https://radix-ui.com) and styled with [Tailwind CSS](https://tailwindcss.com).

Unlike traditional component libraries, mono-ui gives you the **source code** — not a black-box npm package. Use the CLI to copy exactly what you need into your project, then own it completely.

## Quick Start

```bash
npx mono-ui init
npx mono-ui add button
```

## Components

Browse the components in the sidebar. Each one includes:

- Live preview
- Installation command
- Copy-paste source code

## Theme

mono-ui uses a **slate-blue** primary color with sharp `0.375rem` border radius — tuned for clean, professional interfaces.
